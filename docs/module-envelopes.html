(static) freqEnv(frequency, octaves, attack, decay, sustain, release, context)

A frequency envelope. Basically the setup to provide an adsr over a number of octaves.

Parameters:
Name Type Description
frequency Number

the initial frequency
octaves Number

(Optional) the number of octaves of the envelope (1 by default)
attack Number

(Optional) the attack length in seconds
decay Number

(Optional) the decay length in seconds
sustain Number

(Optional) the sustain gain (from 0 to 1)
release Number

(Optional) the release length in seconds
context AudioContext

the optional audio context
