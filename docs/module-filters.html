Parameters:
Name Type Description
type String

the filter type
frequency Number | AudioNode

the frequency in hertzs (can be a number or a signal)

(Optional) options may include:

  • detune: the detune of the frequency in cents (can be a number or a signal)
  • Q: the Q of the filter (can be a number or a signal)
